FAQ: StickNext Saga Rail

Halaman ini dibuat khusus sebagai titik FAQ interaktif untuk pertanyaan umum seputar proyek. Dibuat untuk developer/penikmat yang ingin tahu fitur, navigasi, dan rencana pengembangan.

StickNext Saga Rail bertujuan jadi portal komunitas dengan situs demo kreatif, media visual, dan fitur interaktif untuk anggota serta pengembang.

Tambahkan file HTML baru dan update `navigasi.html` dengan kartu baru sekaligus link. Navigasi utama dijaga sebagai satu-satunya sumber daftar global.

Iya, semua halaman khusus dev (statistik, monitoring) bisa dihubungkan dari `navigasi.html`, agar tidak tersebar di menu umum.

Gunakan `statistik-jaringan.html` (Network Information API), atau gunakan `navigator.onLine`, `fetch` speed test, serta monitoring server backend di lingkungan produksi.

Situs ini tidak mengumpulkan data pengguna secara otomatis. Semua uji hanya berjalan di browser klien dan menggunakan endpoint lokal saat ini.
